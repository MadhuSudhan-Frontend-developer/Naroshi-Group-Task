Login and Signup Page
Project Name

Login and Signup Page using HTML, CSS, and JavaScript

Purpose

This project is used to login and signup users.
New users can create an account using the signup page.
Existing users can login using the login page.

About the Project

This project is made using HTML, CSS, and basic JavaScript.
It uses browser localStorage and sessionStorage to save and verify user data.
The design is simple, clean, and responsive.
The pages work on both mobile and desktop.

Pages Included

Login Page

Signup Page

Technologies Used

HTML

CSS

JavaScript

Features

User signup

User login

Local data storage using browser storage

Session-based login check

Responsive design

Link between Login and Signup pages

How to Use

Open signup.html

Create a new account

After signup, go to login page

Enter the same email and password

Login success message will be shown
